ごあいさつ

Greeting

鹿児島市松元地区に
令和７年５月１日に新しくできた療育施設です。
地域のこどもたちやご家族の
「育ち」や「育て」をサポートし、
なんでも気軽に相談できる場所になりたいと
考えています。

なんでもお気軽にご相談ください🎵

大切にしていること

Values

みらいキッズは、
こどもたちとそのご家族、および地域に寄り添い、
丁寧な支援を行うことを基本理念とした
放課後等デイサービスです。

こどもたちの「つながり」や「学び合い」を
大切にしながら、
「頑張ってよかった！」と感じられるような
支援を目指しています。

  • こども同士が学び合える環境づくり

    小集団の中で友達とたくさん関わり、
    互いに学び合えるようにサポートします。

  • 成長につながる支援の工夫

    一人一人の力に応じた支援の工夫を行い、
    自分の力を存分に発揮できるようにします。

  • ご家族との関係づくり

    ご家族と互いに協力し合い、
    信頼し合える関係づくりを目指します。

  • チームでの統一した支援

    職員間で支援の情報を共有し、統一した
    かかわりを持つことができるようにします。

  • 関係機関との連携

    学校や児童クラブなどと連携し、地域全体で
    こどもたちを支える環境をつくります。

みらいキッズの１日

Day

みらいキッズでは、
活動だけでなく、ここで過ごす時間のすべてが
こどもたちの経験や学びにつながることを
目標にしています。

ご家族と一緒に考えた個別支援計画を基に、
こどもたちの強みや
課題に合わせた遊びや支援の工夫を
毎日丁寧に考えています。
